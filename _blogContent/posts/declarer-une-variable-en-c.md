Syntaxe d'initialisation d'une variable en C++

TYPE NOM (VALEUR);

On peut aussi utiliser la même syntaxe que dans le langage C.

TYPE NOM = VALEUR;

ou encore

TYPE NOM {VALEUR}

Exemple :













    
    #include <iostream>
    using namespace std;
    int main()
    {
     int ageUtilisateur(16);
    return 0; 
    }
